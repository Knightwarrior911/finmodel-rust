---
name: precedent-transactions
description: When the user wants M&A precedent transaction comps, deal multiples, or "what have buyers paid for companies like X".
---
1. Define the screen: sector, size band, and lookback window (default: 5 years; extend to 10 if fewer than 5 relevant deals).
2. Use `research_deal` to find comparable transactions; supplement with `research` for deal terms (announced value, consideration mix, EV/EBITDA and EV/Revenue at announcement, premium to unaffected price).
3. For each deal record: date, acquirer, target, EV, multiple, premium, strategic vs. sponsor buyer. Cite the source for every deal value — no uncited multiples.
4. Note context that skews multiples: cycle timing, synergies baked into price, competitive auctions, distressed sales. Precedents systematically run above trading comps (control premium, typically 20-40%) — say so.
5. Apply the median and interquartile range of deal multiples to the target's metrics for an implied M&A value range.
6. Report: deal table, summary stats, implied range vs. trading comps range, and which precedents are most instructive and why.
